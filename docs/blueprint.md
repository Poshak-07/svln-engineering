# **App Name**: BluePrint Portfolio

## Core Features:

- Project Showcase: Display a detailed view of completed projects, including images, descriptions, and outcomes. Each project should be able to be tagged with a list of technologies used in its creation.
- Services Overview: Clearly list the services offered, detailing what the business specializes in, which matches what the refree provides https://svlnengineering.in
- About Us: An 'About Us' section providing insights into the company's mission, values, and team.
- Contact Information: Provide multiple avenues for contacting the business, including a contact form, email address, and phone number.
- Client Testimonials: Showcase positive feedback from previous clients to build trust and credibility.
- Generative AI introduction tool: Introduce the business using a generated intro paragraph that describes their line of work. Provide controls for adjusting the tone. The LLM will use a tool to retrieve details of the company's line of work from the provided URL.

## Style Guidelines:

- Primary color: Professional blue (#3498DB), representing trust and stability.
- Background color: Light desaturated blue (#EBF5FB), for a clean, modern feel.
- Accent color: Light gray (#A9A9A9) for subtle UI elements and text.
- Font: 'Inter', sans-serif, for a modern, professional look suitable for both headers and body text.
- Use minimalist icons to represent services and skills.
- Employ a clean, grid-based layout to showcase projects and services effectively.
